Détection d’anomalies audio industrielles

Autoencodeur convolutionnel · Log-mel · ROC-AUC

← Retour au portfolio

Contexte

Projet de détection d’anomalies audio industrielles à partir de signaux acoustiques, avec une approche non supervisée basée sur l’erreur de reconstruction.

Objectif

  • Détecter automatiquement des sons anormaux sans labels exhaustifs.
  • Construire un pipeline reproductible (prétraitement → modèle → scoring → évaluation).

Méthode

  • Extraction de caractéristiques : spectrogrammes log-mel.
  • Modèle : autoencodeur convolutionnel entraîné sur données “normales”.
  • Score : erreur de reconstruction (anomalie si score élevé).

Évaluation

  • Métrique : ROC-AUC.
  • Analyse : sensibilité au bruit, seuils, généralisation.

Technologies

Python · PyTorch · NumPy · extraction log-mel
